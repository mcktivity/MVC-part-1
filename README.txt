# hvordan kjører man appen
1. last ned zip fil og pakke ut
2. gå inn i mappen og åpne oblig1.sln med Visual Studio
3. Debug (F5)

# hvordan bruker man appen:
1. registrer som bruker
2. logg inn/velg film
3. velg film/logg inn
4. trykk på kjøp under filmen
5. trykk på "last ned". 
*Ordre historie ligger på "Purchases" og konto info finnes på "Account"
